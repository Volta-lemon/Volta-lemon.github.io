1.3 训练

修改文件目录

export INSTANCE_DIR='/root/pro/GLAD/dataset/MVTec-AD' # 你的，尽量用绝对地址
export OUTPUT_DIR="4000step_bs32_eps_anomaly2" ## 模型输出目录
export ANOMALY_DIR="/root/pro/GLAD/dataset/dtd" # 同上
export CLASS_NAME="all"
export MAX_TRAIN_STEP=4000
export SEED=0

accelerate launch main.py \
    --train=True \
    --pretrained_model_name_or_path="/root/pro/GLAD/stable-diffusion-v1-4" \ # 还有这里需要修改
    --instance_data_dir=$INSTANCE_DIR \
    --anomaly_data_dir=$ANOMALY_DIR \
    --output_dir=$OUTPUT_DIR \
    --checkpointing_steps=500 \ # 多少个保存一下
    --instance_prompt="a photo of sks" \
    --resolution=512 \
    --train_batch_size=2 \
    --class_name=$CLASS_NAME \
    --gradient_accumulation_steps=1 --gradient_checkpointing \
    --use_8bit_adam \
    --mixed_precision="fp16" \
    --learning_rate=5e-6 \
    --lr_scheduler="constant" \
    --lr_warmup_steps=0 \
    --max_train_steps=$MAX_TRAIN_STEP \
    --pre_compute_text_embeddings \
    --seed=$SEED

缺bin文件，可以从它给的地址里下载，如果没下载而是用跑出来的文件加载还是无法加载权重，这是因为我们的文件是.safetensors，更换加载权重的方式即可。

.safetensors 和 .bin 文件都是存储模型权重的方式，但 .safetensors 是一个更优化的格式，提供了安全的序列化和更快的加载。

在main.py的上方写如下代码导入包：from safetensors.torch import load_file

在main.py的264行修改

        )
